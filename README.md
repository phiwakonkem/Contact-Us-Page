# Contact Us Page

A contact form that saves submissions directly to a Firebase Realtime Database — practicing form handling and connecting a static front-end to a live backend service without writing your own server.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat-square&logo=html5&logoColor=white) ![Firebase](https://img.shields.io/badge/Firebase-FFCA28?style=flat-square&logo=firebase&logoColor=black) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat-square&logo=javascript&logoColor=black)

## Features

- Contact form (name, email, number)
- On submit, saves the entry to a Firebase Realtime Database under `contacts`
- Confirmation alert and form reset on successful submission

## Tech Stack

HTML5 · CSS3 · Vanilla JavaScript · Firebase Realtime Database

## Getting Started

```bash
git clone https://github.com/phiwakonkem/Contact-Us-Page.git
cd Contact-Us-Page
```

Open `index.html` directly in your browser. The app connects to its own pre-configured Firebase project, so no setup is required to try it out.

> If you want to point it at your own Firebase project, replace the `firebaseConfig` object in `script.js` with your project's credentials.
